# User Stories

## Overview

User stories describe the features of ExamMaster from the perspective of the users who will interact with the platform. They help define system requirements based on real user needs and expected outcomes.

---

# Student User Stories

## Account Management

### US-001: Student Registration

As a Student, I want to create an account so that I can access the platform and use its learning resources.

### US-002: Student Login

As a Student, I want to log into my account so that I can access my dashboard and continue my learning activities.

### US-003: Password Recovery

As a Student, I want to reset my password so that I can regain access to my account if I forget my login credentials.

### US-004: Profile Management

As a Student, I want to update my profile information so that my account details remain accurate and up to date.

---

# Dashboard and Learning

### US-005: View Dashboard

As a Student, I want to view my dashboard so that I can see my learning progress and recent activities.

### US-006: View Subjects

As a Student, I want to browse available subjects so that I can access content relevant to my examination preparation.

### US-007: Search Subjects

As a Student, I want to search for subjects so that I can quickly find learning materials.

### US-008: View Study Materials

As a Student, I want to access study resources so that I can prepare effectively for examinations.

### US-009: Download Materials

As a Student, I want to download educational materials so that I can study offline when needed.

---

# CBT Practice

### US-010: Start CBT Examination

As a Student, I want to start a CBT practice test so that I can simulate real examination conditions.

### US-011: Answer Questions

As a Student, I want to answer examination questions so that I can test my knowledge.

### US-012: Navigate Questions

As a Student, I want to move between questions during a CBT examination so that I can review my answers.

### US-013: Submit Examination

As a Student, I want to submit my examination when I am finished so that my performance can be evaluated.

### US-014: View Results

As a Student, I want to see my examination results so that I can evaluate my performance.

---

# Performance Analytics

### US-015: View Performance History

As a Student, I want to view my previous examination scores so that I can track my improvement over time.

### US-016: View Subject Performance

As a Student, I want to see my performance by subject so that I can identify areas that need improvement.

### US-017: Monitor Progress

As a Student, I want to monitor my learning progress so that I can stay focused on my academic goals.

---

# Content Manager User Stories

## Resource Management

### US-018: Upload Study Materials

As a Content Manager, I want to upload educational resources so that students can access learning materials.

### US-019: Edit Study Materials

As a Content Manager, I want to update study materials so that content remains accurate and relevant.

### US-020: Delete Study Materials

As a Content Manager, I want to remove outdated materials so that students only access current content.

---

# Question Management

### US-021: Create Questions

As a Content Manager, I want to create examination questions so that students can practice effectively.

### US-022: Edit Questions

As a Content Manager, I want to update examination questions so that errors can be corrected.

### US-023: Delete Questions

As a Content Manager, I want to remove invalid questions so that the question bank remains accurate.

### US-024: Categorize Questions

As a Content Manager, I want to organize questions by subject and topic so that students can easily access relevant content.
