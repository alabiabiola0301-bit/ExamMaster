# Features

## Overview

ExamMaster is designed to provide a complete examination preparation experience for students preparing for WAEC, JAMB, NECO, and GCE examinations. The platform combines learning resources, CBT practice, performance analytics, and administrative tools into a single system.

---

# Minimum Viable Product (MVP)

## Description

The Minimum Viable Product (MVP) contains the core features required to launch the first version of ExamMaster and provide value to students.

# MVP Success Criteria

The MVP will be considered successful when users can:

- Create accounts and log in securely.
- Access study materials.
- Practice CBT examinations.
- View examination results.
- Track performance over time.
- Download educational resources.
- Use the platform without major issues.
